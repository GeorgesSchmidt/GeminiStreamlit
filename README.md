# APPLICATION PERMETANT D'UTILISER GEMENI POUR UNE ANALYSE D'IMAGE OU DE DOCUMENT PDF

On entre un document PDF ou une image avec des mots et on peux poser des questions sur le texte à Gemini. 

Le module `analysePDF.py` analyse une image avec `OCR` en détectant la langue du texte parmi l'anglais, le francais, l'allemand et l'espagnol.
