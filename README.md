# APPLICATION PERMETANT D'UTILISER GEMENI POUR UNE ANALYSE D'IMAGE OU DE DOCUMENT PDF

On entre un document PDF ou une image sur le/laquel(le) il y a du texte et on peux poser des questions sur ce texte à Gemini. 

Le module `analysePDF.py` analyse une image (les pages d'un PDF sont des images) avec `OCR` en détectant la langue du texte parmi l'anglais, le francais, l'allemand et l'espagnol.
